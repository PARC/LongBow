ssize_t longBowDebug_ReadFile ( const char *  fileName,
char **  data 
)

Read data from a file into an allocated array.

Data is read from the specified file into an allocated byte array which must be deallocated by the caller via {stdlib free()}.

For convenience the allocate buffer exceeds the size of the file by one byte, which is set to zero. This permits using the result directly as a nul-terminated C string.

Parameters
[in]fileNameThe name of the file to read from.
[in]dataA pointer to a character pointer that will be updated with the address of the read data.
Returns
The number of bytes read, or -1 if there was an error.

Example:

1 {
2  char *buffer = NULL;
3  size_t numBytesRead = longBowDebug_ReadFile("log.out", &buffer);
4  // use buffer as needed
5 }
