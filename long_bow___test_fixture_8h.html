LongBowClipBoard* longBowTestFixture_GetClipBoard ( const LongBowTestFixture *  testFixture)

Get the clipboard that belongs to the given Test Fixture.

Every LongBow test fixure has an associated "clipboard" that is specific to that fixture and is shared between the test runner setup and tear down functions. Test Runner setup may store things on the test fixture's clipboard, the test fixture may access them and the tear down destroy them.

Parameters
