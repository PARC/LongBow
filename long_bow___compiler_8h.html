#define LONGBOW_CLANG_ATLEAST (   major,
  minor,
  patch 
)    (0)

Compile-time test to determine if the compiler is the GNU C compiler with at least the specfied version, revision and patch levels.

Parameters
[in]majorThe minimum required major version number of the compiler.
[in]minorThe minimum required minor version number of the compiler.
[in]patchThe minimum required patch number of the compiler.
Returns
true If the current compiler matches the requirements.
false If the current compiler fails to match the requirements.

Example:

1 #if LONGBOW_CLANG_ATLEAST(5,0,0)
2  printf("Hello Clang compiler 5.x\n");
3 #endif
