Detailed Description

LongBow Test Case Results.

Copyright
(c) 2013-2014, Xerox Corporation (Xerox) and Palo Alto Research Center, Inc (PARC). All rights reserved.

LongBow Test Cases have expected and actual results. The expected results are typically a statically created instance of LongBowRuntimeResult which is used when the Test Case is executed to compare with the actual results. This permits, for example, a Test Case to indicate that it is expected to induce a specific LongBowEvent. In which case, the actual LongBowEvent must equal the expected event for the Test Case to be considered a success.
